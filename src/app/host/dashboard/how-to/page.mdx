# Com dirigir el joc d'apostes en directe

Esta versió transforma el clàssic format de quiz en una experiència d'apostes en temps real per a reptes físics entre equips. Hi ha quatre equips fixos (roig, blau, verd i groc) i el públic aposta per l'equip que pensa que **perdrà** el repte actual.

## Desenvolupament de cada ronda

1. **Crea una sala** des del tauler. Comparteix l'enllaç o el codi QR perquè l'audiència s'unisca amb un sobrenom.
2. **Selecció de líders** – abans de cada ronda, apunta qui competirà ("Joan vs. Sara"). Esta informació es mostra a totes les persones jugadores.
3. **Votació** – obri la finestra d'apostes perquè els jugadors toquen l'equip que creuen que perdrà. Tothom veu les apostes en temps real.
4. **Acció** – bloqueja les apostes mentre es disputa el repte físic.
5. **Resolució** – marca l'equip que realment ha perdut. El sistema dona 1 punt a cada encert i mostra un panell de transparència amb totes les apostes.
6. **Repetix o tanca** – inicia una nova ronda o finalitza la partida per a mostrar el rànquing definitiu.

## Trucs per a mantindre l'energia

- Mantín les notes dels líders descriptives perquè el públic sàpiga qui competeix.
- Comenta les apostes en directe abans de bloquejar la ronda.
- Utilitza el panell de transparència per a celebrar els bons pronòstics (o per a fer broma amb qui no ha encertat).
